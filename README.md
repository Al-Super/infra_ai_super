# Infra AI Super

Infrastructure repo for the AI Super App.
The infrastructure selected for this project is DigitalOcean.

## Components

- DOCR (Container Registry)
- Droplets (instances)
  - Backend app
  - LLM model app
- MySQL
- Spaces (Bucket Storage)
- IAM
- VPC
